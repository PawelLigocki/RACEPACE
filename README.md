# Race Time Predictor

## Description
Application for runners to calculate pace and predict race times.

## Features
- Pace calculator (min/km)
- Race prediction (Riegel formula)
- Negative split calculator
- Unit tests
- Property-based tests (Hypothesis)

## Tech Stack
- Python
- pytest
- hypothesis

## Run project

Install dependencies:

pip install -r requirements.txt

Run tests:

pytest --cov=app
